## Short Description
This Android calculator application supports basic arithmetic operations (+, −, ×, ÷) as well as exponentiation using a dedicated power button. The app supports chained calculations, error handling for invalid operations such as division by zero, and conversion of results into binary, octal, and hexadecimal number systems. Additionally, it includes a history screen that displays all calculations performed since the app was launched.
